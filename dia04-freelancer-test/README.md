# 🧪 Auditoria de Qualidade — [https://demo.nopcommerce.com/]

## 📌 Objetivo
Avaliar a experiência do usuário e identificar possíveis falhas no sistema.

---

## 🔍 Cenários testados
- Navegação geral
- Busca de produtos
- Acesso a páginas
- Interação com botões

---

## 🐞 Problemas encontrados

### 🔹 Problema 1 — Botão não responde rapidamente
**Descrição:**
Ao clicar no botão, há atraso na resposta.

**Impacto:**
Pode gerar frustração e abandono do usuário.

---

### 🔹 Problema 2 — Texto confuso
**Descrição:**
Algumas mensagens não são claras para o usuário.

**Impacto:**
Dificulta o entendimento e pode causar erro de uso.

---

## 🧠 Análise
O sistema apresenta funcionamento geral adequado, porém possui pontos de melhoria que impactam a experiência do usuário.

---

## 💡 Sugestões
- Melhorar feedback visual dos botões
- Ajustar textos para maior clareza
