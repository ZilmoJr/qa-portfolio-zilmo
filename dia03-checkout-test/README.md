Validar o fluxo completo de compra no sistema.

## 🔹 Caso de Teste — Fluxo de checkout

**Passos:**
1. Realizar login com usuário válido
2. Selecionar um produto
3. Adicionar ao carrinho

5. Clicar em checkout
6. Preencher dados obrigatórios
7. Continuar

**Resultado esperado:**
Usuário deve avançar para próxima etapa do checkout

**Resultado obtido:**
