# 🐞 Bug Report — Mensagem de erro pouco clara no login

## 📌 Descrição
Ao inserir dados inválidos no login, o sistema exibe uma mensagem de erro, porém a mensagem não é clara o suficiente para orientar o usuário.

---

## 🔁 Passos para reproduzir
1. Acessar a página de login
2. Inserir usuário inválido
3. Inserir senha inválida
4. Clicar em login

---

## ✅ Resultado esperado
O sistema deve exibir uma mensagem clara informando que o usuário ou senha estão incorretos.

---

## ❌ Resultado atual
O sistema exibe uma mensagem genérica, dificultando o entendimento do erro pelo usuário.

---

## ⚠️ Severidade
Média

---

## 🌐 Ambiente
- Navegador: Chrome
- Sistema: Windows
- Data: 23/03/2026

---

## 🧠 Observação
A melhoria da mensagem pode reduzir erros do usuário e melhorar a experiência de uso.
